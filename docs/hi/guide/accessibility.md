# एक्सेसिबिलिटी

Vue OTP Pro एक्सेसिबिलिटी को ध्यान में रखकर बनाया गया है।

## ARIA सपोर्ट

- कंटेनर में `role="group"` है
- हर इनपुट में `aria-label` है, जैसे "OTP Input digit 1"
- `ariaLabel` prop से लेबल प्रीफिक्स कस्टमाइज़ करें

## कीबोर्ड नेविगेशन

| कुंजी | क्रिया |
|-------|--------|
| `0-9` | अंक दर्ज करें, अगले पर स्वतः आगे बढ़ें |
| `Backspace` | वर्तमान साफ़ करें या पिछले पर जाएं |
| `Delete` | वर्तमान इनपुट साफ़ करें |
| `ArrowLeft` | पिछले इनपुट पर फोकस ले जाएं |
| `ArrowRight` | अगले इनपुट पर फोकस ले जाएं |
| `Tab` | मानक Tab नेविगेशन |
| `Enter` | सभी भरे हों तो `complete` इवेंट ट्रिगर |
| `Ctrl/Cmd + V` | क्लिपबोर्ड से पेस्ट करें |

## मोबाइल

- `inputmode="numeric"` मोबाइल पर न्यूमेरिक कीबोर्ड दिखाता है
- `autocomplete="one-time-code"` से iOS SMS ऑटोफिल सपोर्टेड
- पेस्ट हैंडलिंग स्वचालित रूप से डिलिमिटर हटाता है
- IME इनपुट (चीनी, जापानी, कोरियाई) कम्पोज़िशन पूरा होने पर मान्य किया जाता है

## अक्षम vs केवल-पढ़ने

| स्थिति | उपयोगकर्ता इंटरैक्शन | दिखावट |
|--------|---------------------|---------|
| सामान्य | पूर्ण इनपुट | मानक |
| `disabled` | कोई इंटरैक्शन नहीं | 45% अपारदर्शिता |
| `readonly` | केवल फोकस/चयन | 70% अपारदर्शिता |
